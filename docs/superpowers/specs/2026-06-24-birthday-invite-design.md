## Структура страницы (сверху вниз)

1. **Фон:** анимированные звёзды (CSS/keyframes), парящие планеты
2. **Хедер:** анимация летящей ракеты (CSS animation)
3. **Заголовок:** «Кирилл приглашает на день рождения!» — крупно, оранжевый
4. **Бейдж возраста:** «4 года» в стиле космической нашивки
5. **Блок с деталями:**
   - 🗓 4 июля 2026
   - ⏰ 13:00
   - 📍 Kids Castle — кнопка «Открыть на Яндекс Картах» (ссылка вставляется позже)
6. **Футер:** «Ждём тебя!»

## Технические детали

- **Стек:** чистый HTML + CSS + JS (без фреймворков)
- **Анимации:** CSS keyframes (ракета, звёзды, парящие элементы)
- **Шрифты:** Google Fonts — беззасечный игровой (например, Nunito или Pacifico для заголовков)
- **Развёртывание:** GitHub Pages (ветка `gh-pages` или `main` в корне)
- **Адаптивность:** да, под мобильные устройства
